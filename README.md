# 4-2_DBMS_assignment
## Personal Calendar
회원가입, 로그인, 일정 추가/수정/삭제, 일정 검색이 가능한 Personal Calendar를 구현하였다. <br>
UI는 java로 구현하였고, Postgresql로 데이터베이스를 관리한다.
